成为见证者

节点不收集你的任何信息,也没有、永远不会有金钱回报—— 它的回报是:平台多一分不敢作假。

方式一 · 打开网页就行(不用懂电脑)

点下面的按钮,你的浏览器就会开始 独立复算平台账本的哈希链和分账规则,每 5 分钟一轮。 保持本页开着,你就在监督我们。

⏳ 正在校验…
首轮校验需要几秒钟
保持本页开启即持续见证 · 停止并退出

方式二 · 电脑绿色版(下载,双击,完事)

单文件绿色软件:不用安装、不写系统、删掉即卸载。 双击后出现一个小窗口,每 5 分钟自动核账一轮,窗口开着就是在见证。

⊞ Windows macOS 🐧 Ubuntu/Linux

macOS 首次打开:右键 → 打开(程序未做付费签名); Linux 解压后双击或 ./chaojizan-witness-linux。程序约 6MB,源码在 witness/go/。

方式三 · 命令行脚本(程序员)

curl -O https://chaojizan.cc/witness.py
python3 witness.py               # 只需要 Python 3.10+,每 5 分钟自动校验一轮

脚本不到 300 行、零第三方依赖,建议读一遍再运行——见证的意义在于你不需要信任任何人,包括我们。网页版的校验代码同样公开:在本页右键"查看源代码"即可审阅。开源仓库(含 Docker 方式)将随公开版一起发布。
