# 税务合规口径(超级赞 Super-Z · 给会计看的系统说明)

> 本文讲清系统里每一分钱的口径与数据出处,**不构成税务建议**;
> 申报科目、税率、优惠政策适用以会计/税务师意见为准。
> 数字口径全部与公开账本(`/ledger/anchors`,witness 哈希链)和
> 三端钱包同源,冲账/调整负数行直接求和即净口径,可交叉验证。

## 一图流:一笔 30 元外卖订单的钱

```
用户实付 30.00 = 菜品 25.00 + 打包 1.00 - 满减 1.00 + 配送费 5.00
├─ 商家应收 25.00(菜品+打包-满减)
│   ├─ 平台佣金 1.25(5%)      → 平台的应税收入(线一)
│   └─ 商家净额 23.75           → 商家的货款,自行纳税(线二)
└─ 配送费+小费 5.00             → 100% 归骑手(线三),平台分文不取
```

## 线一:平台增值税

- **应税收入 = 外卖佣金 + 团购核销服务费**,除此之外平台没有收入:
  配送费/小费是全额过手给骑手的代收代付;超时赔付券、新客券等补贴是
  平台费用(`orders.subsidy_cents` 逐单可查)。
- 申报底稿:`GET /tax/platform-income.csv?period=YYYY-MM`
  (逐笔含冲账负数行,净口径直接求和)。
- **向商家开票(平台服务费电子普票)**:
  - 商家端「对账 → 发票」按自然月申请,金额系统聚合不可改
    (`invoice_requests` 表,pending/issued/rejected 留痕);
  - 管理后台线下开票后回填发票文件链接;
  - 按商家汇总对数:`GET /tax/commission-invoice.csv?period=YYYY-MM`
    (外卖佣金+团购服务费,带发票抬头与税号,开票依据)。
- 分账落地(清单#45)后口径不变:货款路径变为直达商家,佣金留存部分
  依然是平台收入,`profit_sharing_records.commission_cents` 可与本表核对;
  代收代付条目随之消失,报表口径反而简化。

## 线二:商家

- 商家净额是商家自己的营业收入,**平台不代扣代缴**,平台提供结算依据:
  - 平台侧:`GET /tax/merchant-settlement.csv?period=YYYY-MM`(按店汇总);
  - 商家侧:对账页逐日账单 + 对账单导出(清单#55,逐单 CSV)。
- 用户索取餐费发票:由商家自行开具(平台不是货款销售方,开不了这张票),
  用户端已按此提示。
- 过渡期(平台代收付)与分账期(货款直达)只影响资金路径,
  不改变商家的纳税义务。

## 线三:骑手个税

- 骑手所得 = 配送费 + 小费(`rider_earnings` 只追加流水),T+1 批量打款
  零手续费。
- **目标态**:灵活用工平台代发+完税(`services/flexwork.py` 桩已留,
  `.env` 配 `FLEXWORK_APP_ID/SECRET` 启用)——T+1 批次自动走
  `submit_payout_batch` 代发,批次号写进打款凭证,失败走提现 failed 闭环;
  完税凭证由灵工平台出,骑手端入口预留。候选:云账户/好活/薪太软。
- **过渡态(未接入)**:平台人工打款,骑手端收入页展示「完税说明」
  (个税自行申报提示);按人收入证明用
  `GET /tax/rider-income.csv?period=YYYY-MM`。
- 骑手保障金池(每单从佣金计提 `rider_fund_per_order_cents`)是平台的
  费用/负债科目,不是骑手收入;赔付发生时才有支出,公开账本可查。

## 报表清单(管理员导出,均带 BOM,Excel 直接打开)

| 报表 | 用途 |
|---|---|
| `/tax/platform-income.csv` | 平台收入明细(增值税申报底稿) |
| `/tax/commission-invoice.csv` | 按商家汇总佣金(向商家开票依据) |
| `/tax/merchant-settlement.csv` | 商家结算汇总(结算单/对账) |
| `/tax/rider-income.csv` | 骑手所得汇总(灵工对账/收入证明) |

## 待会计确认的三个问题

1. 平台服务费的增值税税目与征收率(小规模/一般纳税人口径);
2. 平台补贴(超时赔付券/新客券,subsidy 口径)的费用列支方式;
3. 骑手保障金池计提的会计处理(计提时挂负债还是支付时进费用)。
